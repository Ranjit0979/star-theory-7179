# star-theory-7179
An online Food Delivery Application having backend REST-API, made in Java Spring Boot, JPA-Hibernate with MySQL database.
# Built with
1. JAVA

2. Spring
3. Spring Boot
4. Spring data JPA
5. Hibernate

# Modules
1. Login Module
2. Costumer Module
3. Restaurant Module
4. Order Module
5. Bill Module
6. Item Module
7. Cart Module
8. Error Module
